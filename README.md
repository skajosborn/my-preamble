# Preamble to the Constitution App

A simple, beautiful React application that displays the Preamble to the Constitution of the United States of America.

## Features

- Built with React and Vite
- Clean, modern design with a gradient background
- Responsive layout that works on desktop, tablet, and mobile devices
- Elegant typography using serif fonts
- Smooth animations and transitions

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or the port Vite assigns).

### Build for Production

Build the app for production:
```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```
